# AwesomeSites

> 很多精彩的网站随着时间推移很有可能会消失在互联网上，所以我建立了这个仓库，将这些值得学习的网站收藏并备份下来。本仓库只是一个目录。特别大的项目（超过100MB）会独立建立保存仓库，中小型项目按照分类存放于`pack`中。

## 使用方法

所有收录的网站都经本人测试可以离线运行，进入网站域名文件夹后开启`live-server`或其他的本地http服务后即可离线运行。
